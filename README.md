# Exclusive
An exclusive website using HTML, CSS, and Javascript

## Pages
- Home Page
- Products Page
- Product Detail Page

## Features
- Dynamic header and footer
- Product listing using JavaScript
- Product detail view
- Responsive design
- Reusable components

## Folder Structure
- components/css → stylesheets
- components/js → JavaScript logic
- components/pages → product pages
- components/images → images
- components/icons → icons

## How to Run
Open `index.html` in a browser.

No build tools or dependencies required.
